# Console-Based Music Player (Java OOP Project)
Project Description
This is a menu-driven console-based Music Player application developed using Java.  
The project demonstrates core Object-Oriented Programming (OOP) concepts:

- Encapsulation  
- Abstraction  
- Inheritance  
- Polymorphism  
- Collections (ArrayList)  

The application allows users to manage songs, create playlists, and control playback operations like play, pause, and stop.

Features

- Add new songs  
- Delete songs  
- Display all songs  
- Create playlists  
- Add songs to playlists  
- Display all playlists  
- Display songs inside a playlist  
- Play song from playlist  


Technologies Used

- Java  
- OOP Concepts  
- ArrayList  
- Scanner

How to Run

1. Open the project in Eclipse or any Java IDE.  
2. Run Main.java.  
3. Use the menu options in the console.
